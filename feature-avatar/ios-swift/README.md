# iOS Swift Avatar Demo

本示例是配合服务端数字人功能的**移动端效果演示 Demo**。

## 启动指引

### 1. 运行移动端 Demo

请先参考 [ios-swift 工程 README](../../ios-swift/README.md) 完成基础配置和环境搭建。

### 2. 启动服务端数字人

在运行移动端 Demo 之前，必须先启动服务端数字人脚本

详细的服务端配置和使用方法，请参考 [服务端 README](../server-python/README.md)。

## Demo 效果

- 默认状态：右上角小窗口显示数字人
- 点击放大窗口，再次点击收起成小窗
- 数字人会根据 AI 回复实时做出表情和口型动作

## 注意事项

- 本 Demo 仅用于展示效果，服务端数字人需要单独启动
